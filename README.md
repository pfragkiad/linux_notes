# linux_notes

Add repository for pip3 (and other)

```bash
sudo add-apt-repository universe
```


## Screen

Create screen (named session)
```bash
screen -S nane_of_session
```

Detach screen
```
<kbd>Ctrl</kbd>-<kbd>A</kbd> + <kbd>D</kbd>
```

List active screens
```bash
screen -ls
```

Reactivate screen
```bash
screen -r
```

Exit screen
```bash
exit
```
